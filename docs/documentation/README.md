# VySol

VySol is a local-first roleplay application for exploring pre-existing fictional worlds with graph-grounded context instead of standard chunk-based retrieval.

VySol is for people who want believable roleplay inside established fictional settings without the context loss that makes standard RAG feel shallow, generic, or inconsistent. It is also useful for local-first, inspectable RAG over private text collections, lore-heavy document sets, continuity work, grounded knowledge bases, and long-form reference material. VySol is still early stage.

## Installation Requirements

Official support currently covers Windows only. Internet access is required for setup and for any cloud-based model or provider you choose to use. A local GPU is optional; VRAM needs depend on what you are doing, e.g. embeddings or extraction workloads with many simultaneous calls. chatting etc. This also varies greatly by KV Cache size and Model Size.
