# Architecture

This section will hold VySol's higher-level architecture notes once they are ready.

The current concept pages explain how individual systems work. This architecture section is reserved for broader material that crosses those system boundaries, especially:

- Architecture Decision Records, for explaining why major technical decisions were chosen over serious alternatives.
- A codebase map, for showing where the main parts of the repo live and how they fit together.

The old full-system diagram has been removed for now because it was trying to show too much in one view and was becoming hard to read. Future diagrams should be added only when they clarify a specific architecture question.
