# Architecture

Architecture has two views because they answer different questions.

- [C4 Container Diagram](c4-container-diagram.md) shows the major running applications, data stores, and external systems.
- [System Flow Diagram](system-flow-diagram.md) shows the feature-level path from source files to chunks, vectors, retrieval results, and model context.

Use the C4 view to understand boundaries. Use the system-flow view to understand the order of work.
