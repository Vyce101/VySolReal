# Quickstart

## Prerequisites
- [Python 3.14.3](https://www.python.org/downloads/release/python-3143/)
- [Node.js 24](https://nodejs.org/en/download)
- [Git](https://git-scm.com/downloads)

## Install (PowerShell)
Open PowerShell in the folder where you want VySol installed, then copy and paste these exact commands:

```powershell
git clone https://github.com/Vyce101/VySolReal.git VySol
cd VySol
```

## Run
Run `run.bat` from the project folder.

On first run, VySol prepares a local Neo4j Community database without installing a Windows service. The downloaded tools are stored under `user/tools`, and Neo4j data plus the generated local connection file are stored under `user/neo4j`.
